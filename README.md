# SlackBot
SlackBotを作るための道具たち
## LICENSE
* MIT LICENSE

## 機能
* Slackのbot機能

## 動作保証環境
* MacOS X 10.10.4
* Ubuntu 14.10
* CentOS 7

## 使い方
* configure-example.jsonをconfigure.jsonにリネームする
* configure.jsonのToken、Room、UserNameを設定する
    * Token: SlackのToken
    * Room: 投稿したい部屋の名前
    * UserName: botが投稿するときのユーザー名
